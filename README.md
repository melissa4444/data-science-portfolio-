## Projects

### Iris Species Classification 
Exploratory data analysis and classification modeling on the Iris dataset using scikit-learn.

**Skills:** Python, Pandas, matplotlib, scikit-learn, Random Forest, feature importance analysis

**Key finding:** Petal dimensions are stronger predictors of iris species than sepal dimensions. 
Model achieved 100% test accuracy.
